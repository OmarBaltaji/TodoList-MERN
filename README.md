## Pre-requisites:
- Make sure you have Node JS installed on your machine.<br>
- Then run the following commands:

```
git clone https://github.com/OmarBaltaji/TodoList-MERN.git
```
```
cd TodoList-MERN
```
```
npm install
```
```
npm run start:dev
```
